# Lecture Snapper

Приложение для автоматического захвата значимых изменений на экране во время лекций.
Вместо видео сохраняет только важные скриншоты и собирает их в PDF.

---

## Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

> **Примечание:** Если `scikit-image` не устанавливается, можно пропустить — приложение будет работать
> с упрощённым алгоритмом сравнения (без SSIM). Остальные пакеты обязательны.

### 2. Запуск

```bash
python main.py
```

---

## Использование

1. **Выберите область** — нажмите «Выбрать…» и нарисуйте прямоугольник мышью на экране (ESC для отмены).
2. **Настройте параметры** (или оставьте значения по умолчанию).
3. Нажмите **▶ Старт** (или F5) — мониторинг запущен.
4. Приложение автоматически сохраняет скриншоты при значимых изменениях.
5. Нажмите **■ Стоп** (или F6) для завершения.
6. Нажмите **📄 Экспорт в PDF** — все кадры соберутся в один файл.

---

## Параметры по умолчанию

| Параметр | Значение | Описание |
|---|---|---|
| Порог изменения | 13% | Минимальная доля изменённых пикселей |
| Интервал проверки | 1.5 с | Как часто анализировать экран |
| Устойчивость | 2.0 с | Изменение должно держаться N секунд |
| Мин. пауза | 8.0 с | Минимум между двумя сохранениями |

---

## Алгоритм детекции

Используется **многоуровневый фильтр**:

1. **Пре-скрининг** — grayscale + resize до 320×240, проверка средней разницы.
   Отфильтровывает мерцание и движение курсора.

2. **Попиксельная разница** — доля пикселей с разницей > 25/255.
   Если < порога → пропускаем.

3. **SSIM** (структурное сходство) — если SSIM > 0.92 → изменение несущественное
   (мелкие артефакты, анимации).

4. **Устойчивость** — изменение должно сохраняться минимум 2 секунды
   (защита от переключения окон туда-обратно).

5. **Минимальная пауза** — между сохранениями не менее 8 секунд.

6. **Антидубликат** — перед сохранением сравниваем с последним сохранённым кадром.
   Если SSIM > 0.88 → это дубликат, не сохраняем.

## Структура проекта

```
lecture_snapper/
├── main.py                        # Точка входа
├── requirements.txt
├── README.md
└── app/
    ├── config.py                  # Настройки по умолчанию
    ├── gui/
    │   ├── main_window.py         # Главное окно
    │   ├── region_selector.py     # Выбор области мышкой
    │   └── preview_widget.py      # Превью последнего кадра
    ├── capture/
    │   ├── screen_capture.py      # Захват экрана (mss)
    │   └── monitor.py             # Фоновый поток мониторинга
    ├── detection/
    │   └── change_detector.py     # Алгоритм детекции изменений
    └── export/
        └── pdf_exporter.py        # Сборка PDF (Pillow)
```
